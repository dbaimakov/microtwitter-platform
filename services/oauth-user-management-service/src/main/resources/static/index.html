MicroTwitter UMS OAuth 2.0

This service signs users in with GitHub and returns a JSON response that contains a UUID session token and a JWT token.

Login with GitHub
  • Browser flow starts at /api/v1/auth/login/github
  • UUID token validation stays at /api/v1/internal/auth/validate?token=...
  • JWT decoding demo is available at /api/v1/auth/jwt/inspect?token=...
